# LDR With Servo Motor
LDR Photoresistor With Servo Motor.

## Prerequisites
1. Arduino Uno R3
2. Arduino IDE
3. LDR (Photoresistor)
4. 10k Ohm Resistor
5. Breadboard / PCB
6. Jumper Wires
